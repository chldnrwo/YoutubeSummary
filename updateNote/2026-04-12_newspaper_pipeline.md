# 📰 업데이트 노트: 나만의 인터넷 신문 파이프라인 개통 및 사용성 개선 (2026-04-12)

이번 업데이트를 통해 단순한 요약 도구를 넘어 개별 인사이트를 융합하여 통찰력 있는 '인터넷 매거진'을 발간하는 핵심 파이프라인이 새롭게 안착되었습니다.

## 🚀 주요 기능 추가 (New Features)

**1. 🤖 AI 카테고리 자동 분류 시스템 ('꼬리표' 기능)**
*   **분석 처리 로직 개선:** 새로운 영상을 분석할 때, Gemini가 원문을 파악하여 자동으로 `경제`, `IT`, `문화`, `그 외` 총 4개의 카테고리 중 하나를 판별하여 JSON으로 반환하도록 `SYSTEM_INSTRUCTION` 프롬프트를 전면 개편했습니다. 
*   **저장 파이프라인 연동:** 추출된 카테고리를 DB(`insights` 테이블의 신규 `category` 컬럼)에 무사히 싣고 저장하도록 백엔드 처리(파싱 및 DB Insert)를 고도화했습니다.

**2. 📰 '나만의 신문 발행' 전용 탭 (Tab 4) 신설**
*   **기간 필터링:** 일자를 지정해 특정 트렌드 내(ex. 최근 일주일) 업로드된 정보만을 정확하게 추려냅니다. (이전에 직접 URL로 분석하여 '업로드 일자'가 없는 과거 데이터는 깔끔하게 필터링 배제하여 에러를 막았습니다.)
*   **카테고리별 병합 조판:** 긁어들인 데이터를 주제별(방)로 묶어 하나의 거대한 텍스트로 합친 뒤, 새로운 '수석 편집장(`SYSTEM_INSTRUCTION_NEWSPAPER`)' 제미나이에게 던져주어 융합된 심층 특집 기사를 써내려가는 파이프라인을 구축했습니다.
*   **영구 저장 보관함:** 한번 발행된 매거진을 일회성으로 날리지 않도록 별도의 신규 `newspapers` 테이블을 구축하여, **[📚 보관중인 신문]** 서랍에서 영구히 빼서 읽어볼 수 있게 만들었습니다.

## 🛠️ 기능 개선 및 버그 픽스 (Improvements & Bug Fixes)

**1. ⬆️ 사이드바 항목 클릭 시 '자동 최상단 스크롤' 구현**
*   마우스를 아래로 휠을 한참 누른 상태로, 왼쪽 사이드바의 어떤 다른 '저장된 분석' 자료를 눌렀을 때 화면 맨 위로 즉시 돌아가지 않아 불편하던 스트림릿의 한계를 자바스크립트 우회 기술로 돌파했습니다.
*   **스트림릿 iframe 캐싱 문제 극복:** 매 클릭 시마다 HTML에 심어둔 자바스크립트가 무시당하는 증상을 막기 위해, 스크립트 코드 속에 `실행 ID` 난수를 심어 매 클릭이 완전히 새로운 스크립트 실행으로 인식되도록 방어 코드를 단단히 쌓았습니다. (강제 0.05초 연속 제어)

**2. 🧼 메뉴 UI 텍스트 간소화 (가독성)**
*   왼쪽 저장된 리스트 탭의 명칭이 `2026-04-11 | 이런저런뉴스` 형태로 너무 길어져 눈이 복잡해지는 문제를 해결. 
*   다시 깨끗하게 `💾 이런저런뉴스...` 단일 포맷으로 원상 복구하고 글자 수를 조정해 시선 분산을 없앴습니다.

**3. 🐛 DB 동기화 안전장치**
*   스트림릿 고유의 렌더링 딜레이(위에서부터 화면을 만들며 DB를 참조하려다 스키마 갱신 코드를 놓쳐버리는 현상)로 인해 발생했던 Database table 에러를 강제 Python 터미널 쿼리 직접 주입을 통해 깨끗하게 치유했습니다.
